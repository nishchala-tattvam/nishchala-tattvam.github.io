# `gstar`

This is a demo library for the project of the C++ course. I chose the first option: star catalog: 
"Design and apply a class hierarchy for storing observational data of astronomical objects".

I decided to have two types of observables: `Star` and `Galaxy`.

Stars have the following property:

- name (string)
- coordinates
- spectral type
- temperature
- age
- radius
- mass

while galaxies have the following property:

- name
- coordinates
- diameter
- type
